Résumé théorique - Chap 1: COMPARER UN LANGAGE DE SCRIPT AVEC UN LANGAGE COMPILÉ

Quiz

  

 
  1. Quelle est la différence principale entre un langage de script et un langage compilé ?
    1.   Un langage de script est plus rapide qu’un langage compilé
    2.   Un langage de script est interprété ligne par ligne, alors qu’un langage compilé est converti en code machine avant l’exécution
    3.   Un langage de script est plus facile à apprendre qu’un langage compilé
    4.   Un langage de script est plus adapté aux applications web qu’un langage compilé
  2. Quel est l’avantage d’un langage compilé par rapport à un langage interprété ?
    1.   Il permet au développeur de mieux contrôler les aspects matériels, comme la gestion de la mémoire et l’utilisation du CPU
    2.   Il permet au développeur de modifier le code sans avoir à le recompiler
    3.   Il permet au développeur d’utiliser une syntaxe plus proche du langage naturel
    4.   ) Il permet au développeur de bénéficier d’une plus grande portabilité du code
  3. Quel est le nom du processus qui consiste à convertir un programme écrit dans un langage interprété en code machine pendant l’exécution ?
    1.   La compilation à la volée
    2.   La compilation juste-à-temps
    3.   La compilation dynamique
    4.   La compilation incrémentale
  4. Quel est le nom du processus qui consiste à vérifier la syntaxe et la sémantique d’un programme écrit dans un langage compilé avant de le convertir en code machine ?
    1.   L’analyse lexicale
    2.   L’analyse syntaxique
    3.   L’analyse sémantique
    4.   La compilation
  5. Quel est le nom du processus qui consiste à exécuter un programme écrit dans un langage interprété ligne par ligne ?
    1.   L’évaluation
    2.   L’exécution
    3.   L’interprétation
    4.   La traduction
  6. Quelle est la première phase de la compilation, qui consiste à découper le code source en unités lexicales ?
    1.   L’analyse lexicale
    2.   L’analyse syntaxique
    3.   L’analyse sémantique
    4.   La génération de code
  7. Quelle est la phase de la compilation qui produit un arbre syntaxique abstrait, qui représente la signification des tokens du code source ?
    1.   L’analyse lexicale
    2.   L’analyse syntaxique
    3.   L’analyse sémantique
    4.   La génération de code
  8. Quelle est la phase de la compilation qui transforme l’arbre syntaxique abstrait en code machine ?
    1.   L’analyse lexicale
    2.   L’analyse syntaxique
    3.   L’analyse sémantique
    4.   La génération de code
  9. Quel est le nom du fichier qui contient le code machine produit par la compilation ?
    1.   Le fichier source
    2.   Le fichier objet
    3.   Le fichier exécutable
    4.   Le fichier binaire
