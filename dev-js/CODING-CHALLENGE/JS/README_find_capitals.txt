Ce texte donne des instructions pour écrire une fonction qui prend en argument une chaîne de caractères (un mot) et qui retourne une liste ordonnée contenant les index de toutes les lettres majuscules dans la chaîne. Le terme "index" fait référence à la position de la lettre dans la chaîne, en commençant par 0 pour la première lettre. Le texte donne également un exemple de test pour la fonction, où la chaîne "CodEWaRs" doit retourner la liste [0, 3, 4, 6], car les lettres majuscules sont aux positions 0, 3, 4 et 6.
