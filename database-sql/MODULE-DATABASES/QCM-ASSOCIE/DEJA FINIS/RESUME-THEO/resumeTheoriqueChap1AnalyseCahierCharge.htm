    3.   Définir les tâches et les activités du projet
    4.   Définir les contraintes du projet

    1.   Elaboration du schéma directeur

    3.   Etude détaillée
    4.   Réalisation
  8. Quel est le type de cahier des charges qui décrit les contraintes techniques et les spécifications du système à réaliser ?
